高血壓非藥物治療

高血壓確診後，應長期堅持高血壓非藥物治療(生活方式干預) ，其中大多數需要長期應用降壓藥物治療，前者是高血壓治療的基石，後者是血壓達標的關鍵，二者相輔相成，缺一不可。

非藥物治療包括提倡健康生活方式，消除不利於心理和身體健康的行為和習慣，達到控制高血壓以及減少其他心血管疾病的發病危險。非藥物治療有明確的降壓效果，如肥胖者體重減輕10kg，收縮壓可下降5～20mmHg；膳食限鹽(食鹽＜6g),收縮壓可下降2～8mmHg。規律運動和限制飲酒均可使血壓下降。對於高血壓患者、正常高值血壓者，以及高血壓易患人群，不論是否已接受藥物治療，均需進行非藥物治療，並持之以恆。

減少鈉鹽攝入

  1. 目標：每人每日食鹽量逐步降至6克。
  2. 措施：
    1. 日常生活中食鹽主要來源為烹飪用鹽以及醃制、鹵制、泡制的食品，應盡量少用上述高鹽食品；
    2. 建議在烹調時盡可能用量具稱量加用的食鹽量，如特製的鹽勺；如普通啤酒瓶蓋去掉膠皮墊後水平裝滿可盛6g食鹽；
    3. 用替代產品，如代用鹽、食醋等；
    4. 宣傳高鹽飲食的危害,高鹽飲食易患高血壓。

合理膳食

  1. 目標：減少膳食脂肪，營養均衡，控制總熱量。
  2. 措施：
    1. 總脂肪量＜30%，飽和脂肪＜10%，每日食油＜25g/d；每日瘦肉類50～100g/d；奶類250g/d；
    2. 蛋類每周3～4個；魚類每周1～3次；
    3. 新鮮蔬菜每日400～500g；水果每日100g；少吃糖類和甜食；
    4. 適當增加纖維素攝入。

規律運動

  1. 目標：強度，中等；頻次，每周5～7次；持續時間，每次持續30分鐘左右，或累計30分鐘。
  2. 措施：
    1. 運動的形式可以根據自己的愛好靈活選擇；
    2. 步行、快走、慢跑、游泳、氣功、太極拳等項目均可；
    3. 運動的強度可通過心率來反映，運動時上限心率=170-年齡；
    4. 對象為沒有嚴重心血管病的患者；
    5. 應注意量力而行,循序漸進；
    6. 一次運動時間不足30分鐘，可以累計計算。

控制體重

  1. 目標：BMI＜24kg/m2；男性腰圍＜90cm，女性腰圍＜85cm。
  2. 措施：
    1. 減少油脂性食物攝入；
    2. 減少總熱量攝入；
    3. 增加新鮮蔬菜和水果的攝入；
    4. 增加足夠的活動量,至少保證每天攝入能量與消耗能量的平衡；
    5. 肥胖者若非藥物治療效果不理想,可考慮輔助用減肥藥物；
    6. 宣傳肥胖的危害,肥胖者易患高血壓和糖尿病。

戒煙

  1. 目標：堅決放棄吸煙，提倡科學戒煙。
  2. 措施：
    1. 宣傳吸煙危害，吸煙有害健康，讓患者產生戒煙願望；
    2. 採取突然戒煙法,一次性完全戒煙，對煙癮大者逐步減少吸煙量；
    3. 戒斷症狀明顯的可用尼古丁貼片或安非他酮；
    4. 避免吸二手煙；
    5. 告誡患者克服依賴吸煙的心理,及懼怕戒煙不被理解的心理；
    6. 家人及周圍同事應給予理解、關心和支持；
    7. 採用放鬆、運動鍛鍊等方法改變生活方式,輔助防止復吸。

限制飲酒

  1. 目標：不飲酒，如飲酒則少量。白酒＜1兩/日；葡萄酒＜2兩/日；啤酒＜5兩/日。
  2. 措施：
    1. 宣傳過量飲酒的危害，過量飲酒易患高血壓；
    2. 不提倡高血壓患者飲酒,鼓勵限酒或戒酒；
    3. 酗酒者逐漸減量;酒癮嚴重者,可借助藥物戒酒；
    4. 家庭成員應幫助患者解除心理癥結,使之感受到家庭的溫暖；
    5. 成立各種戒酒協會,進行自我教育及互相約束。

心理平衡

  1. 目標：減輕精神壓力,保持平衡心理。
  2. 措施：
    1. 保持樂觀性格，減輕心理負擔；
    2. 糾正不良情緒，緩解心理壓力；
    3. 進行心理咨詢；
    4. 音樂療法，及自律訓練；
    5. 氣功等。
